# Habit Tracker

Simple personal web app for tracking up to 15 habits with checkmarks.

## Features
- Up to 15 habits
- Checkmark toggle only
- Remove habits
- Saves to browser local storage

## Run
Open `index.html` in your browser.

## Store on GitHub
1. Create a new empty repository on GitHub.
2. Run:

```bash
git add .
git commit -m "Add simple habit tracker web app"
git branch -M main
git remote add origin <your-github-repo-url>
git push -u origin main
```
